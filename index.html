🔑 配置 GitHub 令牌

编辑功能通过 GitHub API 直接写入仓库里的 qiandou-data.json。请粘贴一个具有该仓库 repo（或 public_repo） 权限的个人访问令牌（PAT）。令牌仅保存在你浏览器的 localStorage 中，不会上传到任何服务器。

获取方式：GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo 权限生成。
